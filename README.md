Инструкция по запуску проекта
1. Подготовка окружения
Убедись, что установлены Docker и Docker Compose. Создайте локальный файл .env на основе шаблона .env.example в корне проекта

2. Сборка и запуск
Выполните команду в терминале bash:
docker-compose up -d --build

3. Переходите на сайт
http://localhost:8080
